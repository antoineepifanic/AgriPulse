Guide de sécurité

Protégez votre compte et vos données sur AgriPulse

Protection du compte

Mot de passe

  • Utilisez un mot de passe fort et unique
  • Changez régulièrement votre mot de passe
  • N'utilisez pas le même mot de passe sur d'autres sites
  • Activez l'authentification à deux facteurs

Connexion sécurisée

  • Vérifiez toujours l'URL du site
  • Ne partagez jamais vos identifiants
  • Déconnectez-vous après chaque session
  • Surveillez les connexions suspectes

Protection des données

Données personnelles

  • Vérifiez les informations partagées
  • Mettez à jour régulièrement vos données
  • Contrôlez vos paramètres de confidentialité
  • Supprimez les données obsolètes

Communication

  • Utilisez la messagerie interne
  • Ne partagez pas d'informations sensibles
  • Signalez les comportements suspects
  • Respectez la confidentialité des autres

Signaler un problème

Que signaler ?

  • Comportements inappropriés
  • Usurpation d'identité
  • Fraude ou arnaque
  • Problèmes de sécurité

Comment signaler ?

  • Utilisez le bouton "Signaler" sur le profil
  • Contactez le support sécurité
  • Fournissez des preuves si possible
  • Restez disponible pour plus d'informations

Bonnes pratiques

Sur le terrain

  • Rencontrez-vous dans un lieu public
  • Informez quelqu'un de vos déplacements
  • Respectez les règles de sécurité
  • Utilisez les équipements de protection

En ligne

  • Mettez à jour vos appareils
  • Utilisez un antivirus
  • Évitez les réseaux Wi-Fi publics
  • Sauvegardez vos données
